# MAAC Ad Audience Segment Export ‚Äì Crescendo Lab Help Center

#### Feature Overview

The **Ad Audience Export** feature allows users to export segmented audience lists as CSV files and automatically convert them into formats compatible with **Meta Ads**, **Google Ads**, and **LINE Ads**. This feature empowers brands to leverage first-party data effectively, enabling them to create remarketing, exclusion, and lookalike audiences, enhancing advertising campaigns' performance.

### Frequently Asked Questions (FAQ)

<details>

<summary>Does the existing export feature provide the same results?</summary>

* Yes, but brands need to format and clean the data manually.
* Advantages of the New Feature:
  * Saves time by eliminating manual formatting.
  * Reduces the risk of errors and improves data matching accuracy.
  * Ensures data security by automatically encrypting PII.

</details>

<details>

<summary>How do I use the exported list on ad platforms?</summary>

* **Google Ads:** Upload the CSV to ‚ÄúCustomer List‚Äù to create remarketing or lookalike audiences. ( [Google's Official Guide](https://crescendolab.zendesk.com/hc/en-us/articles/44903986295961-MAAC-Ad-Audience-Segment-Export))
* **Meta Ads (Facebook):** Upload the CSV to ‚ÄúCustom Audiences‚Äù for remarketing or lookalike targeting. ( [Meta's Official Documentation](https://crescendolab.zendesk.com/hc/en-us/articles/44903986295961-MAAC-Ad-Audience-Segment-Export))
* **LINE Ads:** Upload the CSV to ‚ÄúCustom Audiences‚Äù for precise ad delivery. ( [LINE Ads' Official Guidelines](https://crescendolab.zendesk.com/hc/en-us/articles/44903986295961-MAAC-Ad-Audience-Segment-Export))

</details>

<details>

<summary>How much data is required for an effective ad list?</summary>

* **Google & Meta Ads:** Minimum 1,000 entries, with 50,000+ recommended for best results.
* **LINE Ads:** No official minimum requirement, but larger lists yield better results.

</details>

<details>

<summary>Does the exported file contain Personally Identifiable Information (PII)?</summary>

No, all phone numbers and email addresses are encrypted using **SHA-256** to ensure data privacy.

</details>

<details>

<summary>Does this feature support API export?</summary>

Currently, API export is not supported. Please stay tuned for updates from **Crescendo Lab**.

</details>
