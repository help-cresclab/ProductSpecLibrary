#### ‚ñ∂Ô∏é Setup Steps

{% stepper %}
{% step %}
### Set the Name

Provide a descriptive name for the rich menu.
{% endstep %}

{% step %}
### Choose the Page type

Select Main page (always on) or Multi-page (for switching to the next page).
{% endstep %}

{% step %}
### For main page, choose the Menu type

* Default menu
* Customized menu (Supports specific segments, tags, binding status ‚Äî see: https://crescendolab.zendesk.com/hc/en-us/articles/26951166179609-Tutorial-Personalized-Rich-Menu-NEW)
{% endstep %}

{% step %}
### Choose the default display option

* Expand: automatically displays when a friend opens the account
* Collapsed: automatically hides\
  Then click Next in the upper right corner.

![](../../../.gitbook/assets/32797841269785)
{% endstep %}

{% step %}
### Configure the Menu

Set the menu size, upload images, and define menu text and interactive areas (Open website URL, Send messages, Switch rich menu, Dial phone number).

![](../../../.gitbook/assets/32797828303257)

{% hint style="info" %}
If the rich menu links to a LIFF page, clicks cannot be tracked, and tags cannot be added. Use a third-party redirector for tagging.
{% endhint %}

{% hint style="warning" %}
The "Dial Phone Number" option only allows phone numbers with 1 to 15 digits; symbols like "+" are not allowed.
{% endhint %}

#### Image Specifications for Rich Menus

See also: [TutorialsÔΩúMAAC Materials Size](https://crescendolab.zendesk.com/hc/en-us/articles/4413212138137-Tutorials-MAAC-Materials-Size)
{% endstep %}

{% step %}
### Large Menu (image specs)

* Resolution: 2500 x 1686 pixels
* Max File Size: 1 MB
* File Types: jpg, jpeg, png (transparent png not supported)
{% endstep %}

{% step %}
### Small Menu (image specs)

* Resolution: 2500 x 843 pixels
* Max File Size: 1 MB
* File Types: jpg, jpeg, png (transparent png not supported)
{% endstep %}

{% step %}
### Save as Draft

Save your work as a draft to continue editing later.
{% endstep %}

{% step %}
### Schedule

Set the details for the rich menu schedule.

![](../../../.gitbook/assets/32797841311897)
{% endstep %}

{% step %}
### Permanent

Choose this if the menu should always be displayed.
{% endstep %}

{% step %}
### Scheduled time

Set specific start and end times for when the menu should be displayed.

![](../../../.gitbook/assets/32797841344537)
{% endstep %}

{% step %}
### Finish setup

Once you‚Äôve completed these settings, click Schedule to finish the setup.
{% endstep %}

{% step %}
### Menu states

The menu can have four states:

* Paused: For editing or rescheduling.
* Scheduled: Set to display at a future specified time.
* Ongoing: Currently active.
* Ended: Displayed during the scheduled time and has ended. You can create a copy or delete it.

![](/broken/files/62232ad456155a10cf7093bacfd47c56983ddf23)
{% endstep %}

{% step %}
### Check type

You can check the menu type from "Type".

![](../../../.gitbook/assets/32797841369753)
{% endstep %}

{% step %}
### Important notices

* If there's a time overlap when setting a permanent or scheduled menu, the system will warn you. Save the draft and adjust the overlapping schedule before editing again.
* If the menu has started running, the start time cannot be changed.
* Once saved, the type of rich menu cannot be changed. To modify it, create a copy and reconfigure.
* It takes 3‚Äì5 minutes for the menu to display correctly in the LINE Official Account after scheduling.
{% endstep %}
{% endstepper %}

#### ‚ñ∂Ô∏é Multi-page Setup

{% hint style="info" %}
Reminder: For Multi-page setup, prepare at least 2 rich menu pages. Set the Main page as "Default" or "Customized menu". Then set the second (and subsequent) rich menu page(s) as "Multi-page" to enable proper scheduling.
{% endhint %}

{% stepper %}
{% step %}
### Configure Return to Main Page

To switch from pages two or three back to the main page, set the interactive area type to "Switch Rich Menu" and select the main page from the dropdown.

![](../../../.gitbook/assets/32797828377369)
{% endstep %}

{% step %}
### Edit interactive areas for switching

Return to the main page, edit the interactive area for page switching, and set the type to "Switch Rich Menu". Select the target menu from the dropdown.

![](../../../.gitbook/assets/46886896037657)
{% endstep %}

{% step %}
### Switching to Default or Customized menu

To switch from pages two or three back to the Default or Customized rich menu, set the interactive area type to "Switch Rich Menu" and select Default or Customized from the dropdown.

![](../../../.gitbook/assets/8869226643097)
{% endstep %}
{% endstepper %}
