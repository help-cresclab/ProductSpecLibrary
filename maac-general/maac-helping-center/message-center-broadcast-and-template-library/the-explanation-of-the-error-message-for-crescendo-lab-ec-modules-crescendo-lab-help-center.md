# The explanation of the error message for Crescendo Lab EC Modules – Crescendo Lab Help Center

### If the brand encounters the following error messages while setting up the EC modules, please refer to the relevant explanations below:

{% stepper %}
{% step %}
### Errors Encountered When Setting the Module Token

* Module token does not match the platform: `module token is not for <platform>`
  * Please take a screenshot of the error message and report it to Crescendo partners via the chat window at the bottom right of the MAAC interface.
  * Once Crescendo partners receive your report, they will conduct a check and reply to you as soon as possible.
* Module token expired: `module token expired`
  * Please take a screenshot of the error message and report it to Crescendo partners via the chat window at the bottom right of the MAAC interface.
  * Once Crescendo partners receive your report, they will conduct a check and reply to you as soon as possible.
* Invalid token for form validation： `invalid token`
  * Please refresh the page where you input the module token and check whether the setup page belongs to the Crescendo Lab domain (the domain is cresclab.com).
  * If the issue persists, please take a screenshot of the error message and report it to Crescendo partners via the chat window at the bottom right of the MAAC interface.
  * Once Crescendo partners receive your report, they will conduct a check and reply to you as soon as possible.
* Invalid module token format： `invalid module token format`
  * Please take a screenshot of the error message and report it to Crescendo partners via the chat window at the bottom right of the MAAC interface.
  * Once Crescendo partners receive your report, they will conduct a check and reply to you as soon as possible.
* Invalid module token length：`invalid module token length: <length>`
  * Please take a screenshot of the error message and report it to Crescendo partners via the chat window at the bottom right of the MAAC interface.
  * Once Crescendo partners receive your report, they will conduct a check and reply to you as soon as possible.
{% endstep %}
