# Why do "Broadcasting" or "DPM" fail to be sent? – Crescendo Lab Help Center

Letter Title: \[System Notification] LINE message fatal notification

When a broadcast fails to be sent, we will send a system notification letter to those who have MAAC privileges management. Please check the “System Notification” letter for the possible reasons for the send failure.

#### ▶︎ Related to LINE system and settings

{% stepper %}
{% step %}
### LINE message quota used up

The LINE message quota has been used up; please purchase additional points on the LINE OA backend or contact your LINE agent to complete the action.
{% endstep %}

{% step %}
### LINE Authorization/Authentication Failed

Access Token expired; please contact Crescendo Lab for assistance.

{% hint style="info" %}
Reminder: This situation may also be caused by the LINE OA backend debit failure. Please check with the LINE OA backend to see if this is a billing issue. For billing issues, please contact LINE's Customer Service to help you find out the cause.
{% endhint %}
{% endstep %}

{% step %}
### LINE service error

LINE service error; please contact Crescendo Lab for assistance.
{% endstep %}

{% step %}
### LINE API service timeout

LINE API service timeout; please contact Crescendo Lab for assistance.
{% endstep %}
{% endstepper %}

{% hint style="info" %}
Review: Please refer to ค่าข้อความ LINE คืออะไร? ยอดข้อความไม่เพียงพอต้องทำอย่างไร?\
https://crescendolab.zendesk.com/knowledge/articles/4413231997593/th?brand\_id=1900000273608

Review: Please refer to What is the Access token? What should I do if it expires?
{% endhint %}

#### ▶︎ Related to MAAC system and settings

{% stepper %}
{% step %}
### Segment is still being calculated

The segment "xxx" is still being calculated and is not ready yet. Please make sure the segment is successfully created before re-sending the push message.
{% endstep %}

{% step %}
### Segment has been removed

The segment "xxx" has been removed. Please select a valid segment to re-send the push message.
{% endstep %}

{% step %}
### Message content error

An error was found in the content of the message sent; please contact Crescendo Lab for assistance.
{% endstep %}

{% step %}
### (DPM) Specified Product ID: "xxx"

The product information of the above item is incorrect (most likely: 0 in stock, missing product information, or no longer exists in the Product Feed). Please correct the Product Feed and contact Crescendo Lab to re-send the push message.
{% endstep %}

{% step %}
### (DPM) Product feed content error

Product feed content error: Please contact Crescendo Lab to help fix it before re-sending the push message.
{% endstep %}
{% endstepper %}

#### ▶︎ Others

{% stepper %}
{% step %}
### Unexpected error

An unexpected error occurred; please contact Crescendo Lab for assistance.
{% endstep %}
{% endstepper %}
