# Tutorials｜How to add MAAC users ? – Crescendo Lab Help Center

To facilitate the addition of users to the MAAC platform, brands can use the MAAC permission management feature to add users independently. Please read the following steps and notes carefully to ensure a smooth user addition process.

{% hint style="warning" %}
The invitee's email must be a valid email that can receive messages. Otherwise, the invitation and registration process cannot be completed.
{% endhint %}

### Resend Invitation Email

If the invited user has not yet activated their account, the administrator can click the “Resend” button on the Permission Management page to send the invitation email again.

* If the original invitation is still within the valid period, the old invitation link remains usable.
* Both the old and new invitation links can be used to activate the account as long as they have not expired.

![](../../../.gitbook/assets/49988167669017)
