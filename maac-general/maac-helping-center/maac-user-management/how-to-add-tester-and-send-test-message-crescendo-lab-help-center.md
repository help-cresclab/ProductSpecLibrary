# How to add Tester and send test message – Crescendo Lab Help Center

In features like segmented broadcasts, message templates, and auto-replies, you can designate a specific user as a tester and send a test message to verify formatting and content before scheduling, creating templates, or drafting messages.

{% hint style="info" %}
Adding a tester does not grant backend administrator privileges. To add a backend administrator, follow: Tutorials｜How to add MAAC users ?\
(https://crescendolab.zendesk.com/hc/en-us/articles/36359736768281-Tutorials-How-to-add-MAAC-users)
{% endhint %}

### Setup Steps

{% stepper %}
{% step %}
### Add the user as a LINE follower

Have the user join the brand's LINE official account and become a follower.

![Follower image](../../../.gitbook/assets/39251168693401)
{% endstep %}

{% step %}
### Open Channel settings in MAAC

Go to the MAAC, click on Account > Channel setting.

![Account > Channel setting](../../../.gitbook/assets/48946024515225)
{% endstep %}

{% step %}
### Create a Channel tester

Choose Channel and click on "Create" to add a tester.

![Create channel tester](../../../.gitbook/assets/48946024517529)
{% endstep %}

{% step %}
### Search and add the tester

You can search by LINE display name, LINE UID, or Customer ID. Click Add to successfully add the tester.

![Search and add tester](../../../.gitbook/assets/48946060091929)
{% endstep %}
{% endstepper %}

After adding the tester, return to the editor for segmented broadcasts or auto-replies. After editing the content, click Send Test Message in the top right corner. Select the tester you added and send a test message.

{% hint style="warning" %}
Sending test messages will count towards your message quota. Please check in LINE’s official account to ensure sufficient message credits.
{% endhint %}

![](../../../.gitbook/assets/39251184312473)

![](../../../.gitbook/assets/39251168720281)

{% hint style="info" %}
If you cannot find a user when searching (duplicate display names or special characters), use the contact list to find their LINE UID and then search by LINE UID to add them as a tester.
{% endhint %}

{% stepper %}
{% step %}
### Find the user in Audience

Click Audience on the left sidebar to enter the contact list. Use the search bar to look for the user's LINE display name and compare profile pictures to find the correct user.

![Audience search](../../../.gitbook/assets/48946024523033)
{% endstep %}

{% step %}
### Copy the user's LINE UID

Once you find the user, click on them and copy their LINE UID.

![Copy LINE UID](../../../.gitbook/assets/48946024524569)
{% endstep %}

{% step %}
### Add tester by LINE UID

Return to Account > Permission Management > Tester > Add Tester, change the search filter to LINE UID, and paste the copied UID to find and add the user as a tester.

![Search by LINE UID](../../../.gitbook/assets/39251168788377)
{% endstep %}
{% endstepper %}
