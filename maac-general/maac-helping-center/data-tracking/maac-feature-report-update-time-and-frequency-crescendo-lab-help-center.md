# MAAC Feature Report Update Time and Frequency – Crescendo Lab Help Center

### MAAC Update Time for Reports of Feature

* Real-time update
  * Reports exported from game interactions
  * SMS+ sending detailed record
* Updated every 1–2 hours
  * Broadcast
  * Tracelink
  * Deeplink
  * Rich menu
  * Auto reply
  * Prize management
  * Beacon location trigger
  * Tag
  * SMS+

### MAAC Data Update Time

* Updated daily from 2am to 3am (UTC+8)
  * Product feed
* Updated daily from 3am to 6am (UTC+8)
  * Segments scheduled in the next 24 hours

### MAAC Open API Related Feature Update Time

* Real-time update
  * Get contact list
  * Get message performance report
  * Check import contact task status
  * Get tag list
  * Get tag info
  * Get message template list
  * Get message event list
  * Get LINE Notification Message setting (Only for Taiwan clients)
* Updated every 1–2 hours
  * Get tracelink performance report
  * Get deeplink performance report
* Updated daily from 4am to 6am (UTC+8)
  * Contact Tracking
  * Get segment list
  * Get the segment contact LINE UIDs
* Updated daily from 0am to 3am (UTC+8)
  * API broadcast (MAAC platform)
