# Why can't friends open the MAAC features link? – Crescendo Lab Help Center

#### Mobile device system with LINE APP support

**iOS**

* iOS 13.0 to 13.7: Up to LINE 12.4.0
* iOS 12.0 to 12.5.5: Up to LINE 11.17.0
* iOS 11.0 to 11.4.1: Up to LINE 10.17.0
* iOS 10.0 to 10.3.4: Up to LINE 9.16.1

{% hint style="info" %}
You cannot use the LINE app with iOS 9.3.6 or earlier.
{% endhint %}

**Android**

* Android OS 6.0 to 6.0.1: Up to LINE 11.15.0
* Android OS 5.0 to 5.1.1: Up to LINE 11.5.0

{% hint style="info" %}
- You cannot use the LINE app with Android OS 4.4.4 or earlier.
- Even if you're using Android OS 7.0 or later, you may not be able to install LINE from the app store offered by your device manufacturer or mobile service provider in some cases.
{% endhint %}

Reference: Please refer to the official LINE instructions [LINE usage environment](https://help.line.me/line/ios/pc?lang=en\&contentId=10002433)

#### LINE APP version with MAAC features support

Most MAAC features including deeplink, tracelink, game interaction, rapid referral, and prize management use LINE LIFF to operate.

If your friends have issues experiencing MAAC features, we recommend they use the latest versions of mobile device operating systems and LINE for LIFF apps.

From LINE official instructions, the minimum mobile device operating environment versions are:

* iOS: v10.17.0
* Android: v11.5.0

Reference: Please refer to the official LINE instructions [Recommended operating environment](https://developers.line.biz/en/docs/liff/overview/#operating-environment)
