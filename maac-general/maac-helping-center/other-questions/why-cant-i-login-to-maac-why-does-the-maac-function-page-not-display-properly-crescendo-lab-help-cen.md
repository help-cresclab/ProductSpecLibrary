# Why can't I login to MAAC? Why does the MAAC function page not display properly? – Crescendo Lab Help Center

If you are unable to log in to the MAAC or the MAAC functions are not displayed properly, please check the following points to ensure that the web page is working properly.

{% stepper %}
{% step %}
### Confirm the network connection is normal

Check your internet/network connectivity.
{% endstep %}

{% step %}
### Clear your browser cache

After clearing the cache, confirm if the problem still occurs.

Chrome

* At the top right, click More.
* Click “More tools”, and Clear browsing data.
* At the top, choose a time range. To delete everything, select All time.
* Next to "Cookies and other site data" and "Cached images and files," check the boxes.
* Click Clear data.
{% endstep %}

{% step %}
### Use an incognito/private window

Open an incognito (private) window and try accessing MAAC to see if the issue persists.
{% endstep %}

{% step %}
### Ask your IT staff to check firewall/domain blocks

Please ask your IT staff to check if there are any special domains blocked by the firewall, and set the following domains as a whitelist:

* cresclab.com
* api.cresclab.com
* maac.cresclab.com
* cdn.maac.app
* [s](http://sendbird-upload.s3.amazonaws.com/) endbird-upload.s3.amazonaws.com
* zendesk.com
{% endstep %}
{% endstepper %}
